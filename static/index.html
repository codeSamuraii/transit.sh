Transit.sh

Direct file transfer with your terminal. No subscription, no storage, no additional tool needed.

# Send
curl -v -T <file> https://transit.sh/<some-string>/

Wait for "We are completely uploaded and fine" before next step.
# Receive
curl -JLO https://transit.sh/<some-string>/
# Example
curl -T /music/song.mp3 https://transit.sh/music-for-dad/
curl -JLO https://transit.sh/music-for-dad/

Information

  • While the API allows to transfer large files instantly, my hosting provider's CDN introduces delays and limitations. See step-by-step.
  • Files are not stored by the service at any point and only exist in memory one chunk at a time.
  • This service is hosted as a proof-of-concept for my open source project with no guarantees.

Step-by-step

  1. Run the first command — with your own file and identifier — to send the file.
  2. Wait before starting the download on the receiver side.
    1MB: 5 sec. – 100MB: 1 min. – 1GB: 5 min.
  3. Run the second command — with the correct identifier — to receive the file.
