Transit.sh

Transfer files directly in your terminal. No subscription, no storage, no additional tool needed.

# Send
curl -T <file> https://transit.sh/<some-string>/
# Receive
curl -JLO https://transit.sh/<some-string>
# Example
curl -T /music/song.mp3 https://transit.sh/music-for-dad/
curl -JLO https://transit.sh/music-for-dad

Information

  • Files transiting aren't stored by the service at any point and only exist in RAM one chunk at a time.
  • For files larger than 100MB, the receiver should wait a few minutes before starting the download.
  • Files larger than 2GB are often rejected by the cloud provider's network.
  • This service is hosted as a proof-of-concept for my open source project with no guarantees.

Step-by-step

  1. Run the first command — with your own file and identifier — to send the file.
  2. Wait before starting the download on the receiver side.
    1MB: 5 sec. – 100MB: 1 min. – 1GB: 5 min.
  3. Run the second command — with the correct identifier and no trailing / — to receive the file.
